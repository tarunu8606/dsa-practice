## 🧩 My Approach to Every Problem

1. Read the problem — understand input/output
2. Try it myself for 15 minutes minimum
3. If stuck — think about the pattern, not the solution
4. Write the solution with comments on every non-obvious line
5. Ask: *can this be more efficient?*
6. Commit with a meaningful message
